# About and history of Lua

Lua is a lightweight, high-level, multi-paradigm programming language designed primarily for embedded use in applications.

It was created in 1993 by the members of the Computer Graphics Technology Group (Tecgraf) at the Pontifical Catholic University of Rio de Janeiro, in Brazil.

Lua is used in multiple applications, including but not limited to:

- Roblox
- Garry's Mod
- World of Warcraft
- ComputerCraft (a Minecraft mod)
- Neovim
- PewPew Live

The full list can be found [here](https://en.wikipedia.org/wiki/List_of_applications_using_Lua).
